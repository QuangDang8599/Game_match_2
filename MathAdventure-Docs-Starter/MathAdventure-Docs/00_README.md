# MathAdventure Documentation

## Tổng quan
Tài liệu này là bộ sưu tập đầy đủ các tài liệu thiết kế, kỹ thuật và vận hành cho dự án MathAdventure, một trò chơi giáo dục toán học dành cho trẻ em lớp 2.

## Mục tiêu dự án
- Giúp trẻ học toán thông qua trải nghiệm chơi game vui vẻ.
- Tạo môi trường học tập tích cực, có phản hồi tức thì.
- Xây dựng nền tảng phát triển có thể mở rộng cho các phiên bản sau.

## Cấu trúc tài liệu
1. Tầm nhìn sản phẩm
2. Thiết kế trò chơi
3. Kiến trúc kỹ thuật
4. Hướng dẫn UI/UX
5. Hệ thống gameplay và toán học
6. Kế hoạch kiểm thử và phát triển

## Người dùng mục tiêu
- Trẻ em độ tuổi 6–8
- Phụ huynh và giáo viên cần công cụ học tập bổ trợ
- Nhà phát triển muốn tạo sản phẩm giáo dục tương tác
